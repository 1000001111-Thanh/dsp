LDPC 5G
===============================
This module contains the API for using the GPU-accelerated LDPC coding chain
from the cuPHY library. This includes encoding and decoding as well as
rate matching. Code block segmentation and concatenation are implemented
in Python. Additionally, this module contains a number of utility functions
for example for determining the LDPC base graph, transport block size, etc.
